Delete SPAM Comments from your WordPress website with spambomb.

== Description ==
SPAMBOMB is a WordPress plugin that you can use to remove spam comments with specific words as well as all comments permanently.

== Installation ==
1. Active the plugin
2. Visit Comments->SPAMBOMB DELETE
3. Now you can delete spam comments or all the comments form your website permanently

== Frequently Asked Questions ==

**How do I see the updated Results?**

After selecting an option and clicking on the 'Delete Comments' button, click 'Refresh Page' to view the updated results.

**Which Comments will the plugin delete?**

The plugin will only delete comments with a Pending status to avoid removing Active comments.
